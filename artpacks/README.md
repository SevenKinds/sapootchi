# artpacks — reference material (NOT committed, NOT embedded)

Source/reference art kept out of git (`.gitignore`d): ~2.5 GB total, over
GitHub's limits, and the third-party packs have their own licenses that don't
allow republishing.

- `brand/` — brand kit: logos (horizontal/vertical/icon), editáveis, emoji
  packs (2D/3D/animados/especiais).
- `packs/` — third-party game asset packs for study/prototyping (Tiny Swords,
  Pixel Crawler, Legacy Fantasy, Brackeys platformer).

Anything the game actually ships must be copied into `/assets` (embedded via
`assets/assets.go`) — with a license that permits it.
